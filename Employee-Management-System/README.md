# Employee Management System

## Project Description

This is a Console-Based Employee Management System developed using Python and MySQL. The project allows users to perform basic employee management operations through a menu-driven interface.

## Features

* Add Employee
* View Employee
* Search Employee
* Update Employee
* Delete Employee
* Menu Driven Program

## Technologies Used

* Python
* MySQL
* MySQL Connector

## Python Concepts Used

* Functions
* Dictionary
* Loops
* Exception Handling
* CRUD Operations

## Database

Database Name: `employee_db`

Table Name: `employee`

## How to Run

1. Install Python.
2. Install MySQL.
3. Install the required package:

   pip install mysql-connector-python
  
4. Create the `employee_db` database and `employee` table.
5. Update the MySQL username and password in the Python file.
6. Run:

   python employee_management.py
